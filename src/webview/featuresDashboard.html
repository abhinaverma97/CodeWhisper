🎙️ CodeWhisper Dashboard

Complete guide to CodeWhisper's natural language features

How to use CodeWhisper

Press Ctrl+Shift+V (or Cmd+Shift+V on Mac) to activate the microphone. Wait for the beep, speak your command naturally, and pause for 1 second. The AI will automatically detect when you stop speaking and process your command.

🧠 AI Code Understanding

📖

Explain Code

Select any code snippet or function and ask the AI to explain it in plain English.

Try saying: "Explain what this python method does" "How does this auth code work?"
🔧

Fix Errors

Diagnoses the nearest error or fixes bugs in the code you have selected.

Try saying: "Fix this error" "Why is this throwing a type error?"
📋

Summarize File

Reads through the entire active file and gives you a high-level summary of its purpose.

Try saying: "Give me a quick summary of this file"
🚨

Security Audit

Scans the current file or selection for common security vulnerabilities.

Try saying: "Scan this page for security issues"

✍️ Writing & Editing Code

✨

Generate Code

Describe what you want to build and the AI will write the code and insert it at your cursor.

Try saying: "Write a function that fetches user data"
📄

Create New File

Opens a new unsaved editor for the language you specify.

Try saying: "Create a new typescript file"
💬

Comment Code

Adds helpful, descriptive comments explaining the selected code blocks.

Try saying: "Add comments to this method"
🧪

Generate Tests

Creates comprehensive unit tests for the selected function in a new tab.

Try saying: "Create unit tests for the login function"

🧭 Editor Navigation & Reading

👁️

Read Code

Reads the current line or the entire file aloud using Text-to-Speech.

Try saying: "Read this line" / "Read the whole file"
🚦

Jump to Lines / Symbols

Navigate instantly without needing to search or scroll.

Try saying: "Jump down to line 42" / "Go to handleSubmit"
🐞

Read Errors

Lists all compiler errors and warnings in the current file aloud.

Try saying: "Find all the problems in this file"
⚙️

Editor Actions

Perform common VSCode actions without touching the keyboard.

Try saying: "Save the file" / "Format document" / "Open terminal"

🔔 Sound Alerts Legend

CodeWhisper uses procedural Web Audio to give you spatial awareness of background events.

📉 Low buzzing drone — New Error Detected
🟡 Double beep — New Warning Detected
✨ Ascending chime — All errors cleared!
💾 Soft click — File Saved
🛑 Bell ding — Debugger Breakpoint Hit
✅ Happy arpeggio — Test Passed
❌ Sad descending — Test Failed
📝 Soft pop — Code Suggestion Generated
